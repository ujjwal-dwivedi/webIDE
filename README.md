# Web-Based IDE

A full-stack web-based IDE that allows users to write, edit, and execute HTML, CSS, and JavaScript code in real time within a unified environment.

## Features

- Real-time execution of HTML, CSS, and JavaScript
- Multi-file editor with separate sections for HTML, CSS, and JS
- Live preview with dynamic code integration
- JWT-based authentication and protected routes
- Create, save, retrieve, and delete projects
- Persistent data storage using MongoDB
- Responsive and modern UI with React.js and TailwindCSS

## Tech Stack

### Frontend
- React.js
- TailwindCSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT (JSON Web Token)
